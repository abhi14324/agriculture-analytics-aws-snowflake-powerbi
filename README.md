# Agriculture Analytics using AWS, Snowflake and Power BI

## Project Overview

This project is a cloud-based agriculture analytics dashboard developed using Power BI, AWS, and Snowflake to analyze agriculture-related environmental and crop performance data.

The dashboard focuses on agricultural trend analysis across multiple dimensions including year, season, crop type, and location.

The project demonstrates cloud-based reporting workflow where data is processed through AWS and Snowflake before visualization in Power BI.

The final dashboard was also published on Power BI Service.

---

## Objective

The main objective of this project is to analyze agriculture data and understand how environmental and production factors vary across crops, seasons, and locations.

The dashboard helps identify:

* rainfall trends
* humidity patterns
* temperature variation
* crop yield distribution
